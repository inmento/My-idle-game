WHAT IS NIGHTMARE MODE?

Each Nightmare Run gives you a set of random targets — specific colors must reach specific stat thresholds. Complete the targets to finish the run and start the next one. Runs are endless: each cycle the targets grow harder. There is no finish line.

HOW TO START

Tap the GOAL button to open the Mastery menu, then press START NIGHTMARE RUN. Your world resets, but your permanent upgrades carry over.

GOD MODE EXIT

Struggling? You can buy God Mode Access from the Shop for 100 CR. Once purchased, a GOD MODE EXIT button appears inside the nightmare overlay — press it to leave the run without penalty.

CHANGED YOUR MIND?

After using God Mode Exit, a free UNDO toggle appears in the Shop. Tap it to revert all changes made since God Mode was activated and resume your nightmare run exactly where you left off — same targets, same cycle.
